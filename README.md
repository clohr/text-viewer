# Elm Text Viewer

## Overview
Uses [Prism.js](https://prismjs.com/) and [Elm](https://elm-lang.org/) to create a basic text viewer application.

## Initialize Project
1. `npm i --global uglifyjs elm-test`
2. `elm install`

## Build Project
`make`

## Run Web Server
`make serve`

## Run Tests
`make test`
